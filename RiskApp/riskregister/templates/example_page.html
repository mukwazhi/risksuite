How to Use the New Base Templates

This is an example page showing how to use the new base template structure.

Available Base Templates:
  • base.html - Includes top navigation and side navigation (use for dashboard-style pages)
  • base_nosidebar.html - Includes only top navigation (use for full-width pages)
Navigation Components:
  • includes/topnav.html - Top navigation bar with user menu and main links
  • includes/sidenav.html - Sidebar navigation with workflow and quick stats
